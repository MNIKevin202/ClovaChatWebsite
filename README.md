# ClovaChat Website

Initial static website for ClovaChat, designed to match the dark neon desktop app.

## Local Preview

```bash
python3 -m http.server 4173
```

Open `http://localhost:4173`.

## CapRover

This repo includes:

- `captain-definition`
- `Dockerfile`
- `nginx.conf`

CapRover can build and serve the static site with nginx on port 80.
